# Animal Dash

A cheerful, browser-based animal racing game designed for young children.

## Play locally

Open `index.html` in a browser. For the most reliable experience, serve the folder with a tiny local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- Left/right arrow keys: move between lanes
- Space: jump
- Touch buttons appear on phones and tablets

## GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.

The game has no external dependencies, no tracking and no server-side code. Progress and best times are stored only in the player's browser. Once loaded from GitHub Pages, it can also work offline and may be installed to a phone or tablet home screen.
